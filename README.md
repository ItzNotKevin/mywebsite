# kevinwxli.dev

Personal portfolio website built with a macOS desktop aesthetic.

## Stack

- **Framework:** Next.js (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Language:** TypeScript

## Features

- macOS-style UI with Notes, Terminal, and Finder windows
- Finder window showcasing projects (Tandem, Resonance)
- Animated dock with social links and CS Webring integration
- Scroll-driven transitions between sections

## Projects

- **Tandem** — AI study companion built at Hack Canada 2026
- **Resonance** — Music recommendation app with swipe-based discovery
